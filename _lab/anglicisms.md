# Запрещённые англицизмы

Этот файл используется скиллом content-validator (правило A1) для проверки контента. Если слово есть в таблице — использовать русский аналог. Англицизм допускается в скобках при первом упоминании.

## Таблица замен

| Англицизм | Русский аналог | Контекст / пример |
|---|---|---|
| sink / sinks | механизм удержания | velocity sink → механизм удержания токенов |
| velocity | скорость обращения | token velocity → скорость обращения токена |
| staking | стейкинг | **допустимо** как устоявшийся термин без русского аналога |
| lock / lock-up | блокировка | governance lock → блокировка для голосования |
| supply | предложение | total supply → общее предложение |
| burn | сжигание | token burn → сжигание токенов |
| revenue share | распределение дохода | revenue share model → модель распределения дохода |
| holder | держатель | token holder → держатель токенов |
| market cap | капитализация | market cap → рыночная капитализация |
| transaction volume | объём транзакций | — |
| holding reward | награда за удержание | — |

## Допустимые англицизмы (нет русского аналога)

- staking, DeFi, NFT, AMM, DEX, CEX, TVL, APY, APR
- bonding curve, vesting, airdrop, slashing
- governance (в контексте DAO)
- TGE (Token Generation Event)
- EIP-1559, PoS, PoW

## Правило использования

1. При первом упоминании: русский аналог + англицизм в скобках
   - "скорость обращения (velocity)"
   - "механизм удержания (velocity sink)"
2. Далее по тексту: только русский аналог
3. В формулах: допускается использование общепринятых обозначений (V, M, PQ)
4. В заголовках: только русский аналог
